## 修正したい点

ここです。

```jsx
<input type="text" value={keywordAge} onChange={(e)=>setKeywordAge(Number(e.target.value))} />
```

これは今回少しズレます。

### なぜか

`keywordAge` は input に入る途中の値なので、まずは **文字列のまま持つ** のが自然です。
今の書き方だと、入力するたびに `Number(...)` してしまいます。

たとえば、

* 空文字 `""` を消した瞬間
* 入力途中の値

の扱いが不自然になりやすいです。

なので、`onChange` ではこうします。

```jsx
onChange={(e) => setKeywordAge(e.target.value)}
```

そして、**更新するときにだけ** `Number(keywordAge)` にします。

## 修正版

```jsx
import { useState } from "react";

const initialUsers = [
  { id: 1, name: "Taro", age: 23 },
  { id: 2, name: "Hanako", age: 45 },
  { id: 3, name: "Jiro", age: 30 },
];

export default function App() {
  const [users, setUsers] = useState(initialUsers);
  const [selectId, setSelectId] = useState(1);
  const [keywordAge, setKeywordAge] = useState("");

  const editSet = () => {
    if (!keywordAge.trim()) return;

    setUsers((prev) =>
      prev.map((user) =>
        user.id === selectId
          ? { ...user, age: Number(keywordAge) }
          : user
      )
    );

    setKeywordAge("");
  };

  return (
    <>
      <select
        onChange={(e) => setSelectId(Number(e.target.value))}
        value={selectId}
      >
        <option value="1">1</option>
        <option value="2">2</option>
        <option value="3">3</option>
      </select>

      <input
        type="text"
        value={keywordAge}
        onChange={(e) => setKeywordAge(e.target.value)}
      />

      <button onClick={editSet}>編集</button>

      <ul>
        {users.map((u) => (
          <li key={u.id}>
            {u.id}/{u.name}/{u.age}
          </li>
        ))}
      </ul>
    </>
  );
}
```

* 実際に保存する値
