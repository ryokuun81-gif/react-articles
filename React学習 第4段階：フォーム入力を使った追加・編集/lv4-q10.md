
# Q10

## テーマ

**name と age を同時に編集する**

## 初期データ

```jsx
const initialUsers = [
  { id: 1, name: "Taro", age: 23 },
  { id: 2, name: "Hanako", age: 45 },
  { id: 3, name: "Jiro", age: 30 },
];
```

## 基本形

```jsx
import { useState } from "react";

export default function App() {
  const [users, setUsers] = useState(initialUsers);
  const [selectedId, setSelectedId] = useState("1");
  const [inputName, setInputName] = useState("");
  const [inputAge, setInputAge] = useState("");

  return (
    <>
    </>
  );
}
```

## 問題

次の動きを作ってください。

* `select` で編集したいユーザーの id を選ぶ
* 名前 input に新しい name を入れる
* 年齢 input に新しい age を入れる
* 「更新」ボタンを押す
* 選ばれた id のユーザーの `name` と `age` を両方更新する
* 更新後は両方の input を空に戻す

---

## 条件

* `inputName` が空文字または空白だけなら更新しない
* `inputAge` が空文字または空白だけなら更新しない
* `age` は数値で保存する
* `map` を使って1件更新する
* `selectedId` は文字列でも数値でもOK

---

## 一覧表示

```jsx
<li key={user.id}>
  {user.id} / {user.name} / {user.age}
</li>
```
---

## ヒント

更新はこの方向です。

```jsx
setUsers((prev) =>
  prev.map((user) =>
    user.id === Number(selectedId)
      ? { ...user, name: inputName, age: Number(inputAge) }
      : user
  )
);
```
